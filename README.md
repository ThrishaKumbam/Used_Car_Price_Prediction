# Used Car Price Prediction System

This project is a machine learning based web application designed to predict the price of used cars using historical vehicle data.

The system analyzes various car attributes such as manufacturing year, mileage, brand, engine capacity, and transmission type to estimate the market price of a used vehicle.

## Project Overview

The goal of this project is to build a predictive model that can help estimate the price of used cars using machine learning techniques. The project integrates data preprocessing, model training, and a web interface for prediction.

## Features

• Predict used car prices based on vehicle attributes  
• Machine learning based prediction model  
• Dataset preprocessing and analysis  
• Web interface for entering car details  
• SQL database integration for storing data

## Dataset

The dataset contains multiple attributes of used vehicles including:

• Car model  
• Year of manufacture  
• Mileage  
• Fuel type  
• Engine capacity  
• Transmission type  
• Price

These features are used to train machine learning models for price prediction.

## How to Run the Project

Clone the repository
git clone <repo-link>

Navigate to project
cd USED_CAR_PRICE_PREDICTION

Install dependencies
pip install -r requirements.txt

Run the Django server
python manage.py runserver

Open in browser
http://127.0.0.1:8000/
